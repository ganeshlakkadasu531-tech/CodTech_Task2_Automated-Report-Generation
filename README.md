## Project Overview

The script reads data (for example, student marks) from a CSV file, performs calculations, and generates a well-formatted PDF report with summaries and tables automatically.

It demonstrates the practical use of:

* **Data analysis** using `pandas`
* **PDF creation** using `fpdf2`
* **Automation** of repetitive reporting tasks

---

##  Requirements

* Read data from a file (CSV)
* Analyze the data (count, average, highest, lowest, pass percentage, etc.)
* Generate a formatted PDF report
* Use `FPDF` or `ReportLab` libraries
* Deliverables: Python Script + Sample PDF
